# Prolog Tic-Tac-Toe KB

This is a minimal Prolog KB implementing a solver for response to a game of tic-tac-toe based on Newell and Simon's 1972 tic-tac-toe program as described by Crowley and Siegler.<sup>\[1]</sup>

** This KB is currently unfinished, and the "Block opponent fork" strategy is not yet implemented, nor has any public API been implemented. The KB is currently a stateless reasoning engine. **

# References

1. Kevin Crowley, Robert S. Siegler (1993). "Flexible Strategy Use in Young Children's Tic-Tac-Toe". Cognitive Science. 17 (4): 531–561. doi:10.1207/s15516709cog1704_3. https://onlinelibrary.wiley.com/doi/10.1207/s15516709cog1704_3
